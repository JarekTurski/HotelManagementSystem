HotelManagementSystem is a simple API exposing several features:

- Get all rooms with optional possibility to filter by name, size and availability
- Get single room by id
- Create hotel room
- Update Details of hotel room
- Mark Room as occupied/not available when:
    * Room is booked, and we have a guest(s) inside
    * Room is during cleaning by the hotel staff
    * Room is locked due to maintenance or repairs
    * Room is manually locked by the hotel staff, to secure it for premium/special guest During making room as not available add possibility to provide some additional details


 Quickstart:
 1. clone repository to you local machine
 2. start the app (no additinal config required)
 3. go to swagger/index.html and test it :)
